# FuCubeMC-Automatic

一个MCSManager自动化前端系统(在建)

此项目基于 Python 开发，以 sqlite 作为数据库的MCSM一键对接网站

![maven](https://img.shields.io/badge/python-3.8%2B-blue)

An automatic controller for MCSManager

# 启动服务

```shell
python server.py
```
